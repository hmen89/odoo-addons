Automatic Logout After Inactivity

Guard your Odoo Session

If you are the kind of person who logs in and never log out, this is for you.

Configurable

Each user has a different idle timer

By default users have 15 minutes of inactivity.

Each user can have their own inactivity timers, defined by seconds.

Be Careful

If you were working and go for a coffee, please press SAVE before you leave.
